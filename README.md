# Moon

Linguagem de Programação Moon.<br>
Stable 6.6.6

```
funcao fibonacci(n, _a, _ba)
  se n == 0 entao
    retorne _a
  senaose n == 1 entao
    retorne _b
  fim
  retorne fibonacci(n - 1, _b, _a + _b)
fim

local f = fibonacci(20, 0, 1)
imprimir(cadeia.repetir(f .. "? E dai?\n", f))
```
